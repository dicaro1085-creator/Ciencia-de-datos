SISTEMA DE GESTIÓN DE CONTACTOS
Este sistema ha sido desarrollado como parte de la evaluación del Módulo 2 para permitir el registro, edición, eliminación y búsqueda de contactos de manera eficiente. El objetivo principal es gestionar información personal de clientes de forma segura y organizada.

El proyecto cumple con las siguientes funcionalidades generales:
Registro de contactos: Permite agregar nombre, teléfono, correo y dirección
Edición y eliminación: Gestion de datos de contactos existentes
Busqueda: Localización de registros por nombre o número de teléfono.

Para garantizar la escalabilidad y calidad del código, se aplicaron los siguientes detalles técnicos:
Programación Orientada a Objetos(POO): Uso de clases para representar contactos, aplicando principios de encapsulación
Estructura de datos: Organización de la información mediante listas y diccionarios
Arquitectura: Código organizado por métodos y atributos para facilitar la legibilidad y mantenimiento.
